diambil adalah data dari website
[https://pemilu2024.kpu.go.id](https://pemilu2024.kpu.go.id/)

## Disclaimer

- Data ini boleh digunakan untuk kepentingan apapun, selama tidak melanggar
  hukum.
- Data dari website KPU masih dalam proses, belum lengkap. Jadi, data yang
  diambil juga belum lengkap.
- Data yang ada di database ini akan diupdate untuk mengikutin perkembangan data
  di website KPU.

## Database

> Catatan
>
> - Data akan terus kami update seiring dengan perkembangan data di website KPU,
>   dan dimasukkan ke dalam database ini. Sedangkan file yang diunggah ke Google
>   Drive belum bisa diupdate secara otomatis, jadi untuk mendapatkan data yang
>   paling terkini, disarankan untuk menggunakan database langsung.
>
> - Acuan ada pada kolom `updated_at` di setiap tabel, yang menunjukkan kapan
>   data terakhir ditarik dari website KPU. Sedangkan kolom `ts` adalah
>   timestamp dari website KPU itu sendiri.
>
> - Selama proses development, database ini bisa berubah struktur, jadi pastikan
>   untuk selalu memeriksa dokumentasi ini.

## API

Selain itu, data ini juga bisa diakses melalui API. Namun baru terdapat satu
endpoint yang bisa diakses dan baru berupa data TPS untuk Pemilihan Presiden dan
Wakil Presiden.

API dapat diakses di
`https://pantau-api.razanfawwaz.dev/api/v1/ppwp?limit=10&offset=0`

Parameter yang bisa digunakan adalah:

- `limit` - Jumlah data yang ingin diambil
- `offset`- Data yang ingin diambil dimulai dari data keberapa

Pastikan untuk menggunakan parameter `limit` dikarenakan jika tidak digunakan,
maka data tidak akan muncul.
