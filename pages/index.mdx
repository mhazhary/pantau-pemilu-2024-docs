## Catatan

- Data ini boleh digunakan untuk kepentingan apapun, selama tidak melanggar hukum.
- Data dari website KPU masih dalam proses, belum lengkap. Jadi, data yang diambil juga belum lengkap.
- Data yang ada di database ini akan diupdate untuk mengikutin perkembangan data di website KPU.
